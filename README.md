# Проект (частина Б) - УСПАДКУВАННЯ. АБСТРАКТНІ КЛАСИ. ПОЛІФОРМІЗМ
## Опис
Цей проект реалізує систему управління бібліотекою відеоігор. Застосунок дозволяє:
- Додавати та видаляти ігри (звичайні та онлайн).
- Переглядати деталі гри та досягнення.
- Запускати, паузити та завершувати ігрові сесії.
- Клонувати та порівнювати ігри.
- Демонструвати поліморфізм через абстрактні класи та інтерфейси (`ICloneable`, `IComparable`, `IPlayable`).
- Переглядати магазин цифрових та фізичних ігор.
## Особливості реалізації
- Використано **абстрактні класи** та **інтерфейси** для демонстрації поліморфізму.
- Всі методи отримали реальні реалізації, замість заглушок.
- Меню програми логічно структуроване та відповідає предметній області.
- Усі unit-тести пройдено успішно.
## Unit-тести
Проект містить 15 unit-тестів, що перевіряють функціональність основних методів, роботу з досягненнями та ігровими сесіями, а також коректність реалізації інтерфейсів.
## Висновки
У результаті виконання роботи було створено стабільний та функціональний застосунок для управління бібліотекою відеоігор. Проект демонструє правильне використання абстрактних класів, інтерфейсів та поліморфізму. Меню та функціонал програми дозволяють коректно виконувати всі операції: додавання ігор, управління сесіями, роботу з досягненнями та магазином. Unit-тести та функціональні перевірки підтвердили надійність та коректність реалізації.
